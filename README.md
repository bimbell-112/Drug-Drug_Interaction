# Drug-Drug Interaction Predictor 💊🤖

## Overview
This is a lab-based project focused on predicting adverse drug effects using Graph Neural Networks (GNN) for drug-drug interaction analysis. 

## Features
* Analyzes complex drug interaction networks.
* Utilizes GNN-based machine learning models to identify potential side effects.

## Project Structure
All core project files, including training scripts (`train.py`), prediction models (`predict.py`), and data processors, are located in the `drug_interaction_gnn_25K` directory.
